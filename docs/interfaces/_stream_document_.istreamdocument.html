close

  • close(): string | Buffer
    • Defined in stream-document.ts:51

    Closing a stream document will prevent any further writes to the document. If the object was instantiated with a file path, close will write to this file. If "new" was called without any args, the document is written to a nodejs buffer, close will return the buffer.
